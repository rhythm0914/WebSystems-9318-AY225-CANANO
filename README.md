# Bacoor City LGU Website

A professional Local Government Unit (LGU) website for Bacoor City, providing information and services to citizens.

## Project Overview

This website simulates a real LGU portal with public-facing services, implementing all required JavaScript functionalities and following professional government design standards.

## Features

### Required Pages
1. **Home** - City overview, highlights, quick links, emergency contacts
2. **About** - History, vision, mission, city officials, facts
3. **Departments** - All city departments with search and filter functionality
4. **Services** - Online services with interactive forms and validation
5. **Announcements** - Dynamic news feed with filtering and modal details
6. **Contact** - Citizen feedback form with validation

### JavaScript Functionalities
- ✅ Dynamic announcements/news feed with filtering
- ✅ Search and filter for departments/services
- ✅ Interactive service request forms with validation
- ✅ Modal dialogs for announcements and advisories
- ✅ LocalStorage for user preferences and recent views
- ✅ Theme toggling (light/dark mode)
- ✅ Form draft saving
- ✅ Service request history
- ✅ Responsive design with mobile navigation
